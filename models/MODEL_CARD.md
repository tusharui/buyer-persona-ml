# Model Card — Buyer Persona ML

## Model Summary
Unsupervised customer segmentation pipeline using KMeans clustering on RFM + behavioral features with PCA dimensionality reduction.

## Intended Use
Segment customers into behavioral personas for targeted marketing, retention campaigns, and business strategy.

## Model Architecture
- **Feature extraction:** 14 customer-level features (RFM + behavioral)
- **Feature selection:** Correlation threshold (|r| > 0.85) + variance filter
- **Dimensionality reduction:** PCA (95% explained variance)
- **Clustering:** KMeans (k=4, tuned via silhouette score)

## Performance
| Metric | Value |
|--------|-------|
| Silhouette Score | 0.13 (moderate overlap — expected for synthetic data) |
| Davies-Bouldin Index | 1.99 |
| Stability (ARI) | 0.74 ± 0.16 |

## Data
- **Source:** 10,000 synthetic invoice transactions
- **Customers:** 1,000 unique
- **Time range:** 1 year
- **Features used:** Recency, Frequency, Monetary, AvgBasketSize, PurchaseInterval, WeekendRatio, NightRatio, DiscountUsage, ReturnRate, ProductDiversity

## Limitations
- Trained on synthetic data — real-world performance may differ
- Silhouette score is below 0.3, indicating cluster overlap
- Assumes customer behavior is stable over time (no concept drift handling)

## Personas

## Fairness
- No demographic features used — no bias from age, gender, or location
- All customers treated equally by the model
